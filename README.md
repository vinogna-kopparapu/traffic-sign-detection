# traffic-sign-detection
a computer vision techniques to automatically detect and classify traffic sign


Features:
Image Classification: Classifies various traffic signs, such as stop signs, speed limits, and warning signs.
Real-time Detection: Detects traffic signs from live video feeds or static images.
Pre-trained Models: Utilizes pre-trained convolutional neural networks (CNN) for efficient and accurate classification.
Data Augmentation: Implements data augmentation techniques to improve model robustness and accuracy.
Model Evaluation: Includes performance metrics such as accuracy, precision, recall, and F1 score.
Tech Stack:
Python, OpenCV, TensorFlow/Keras, scikit-learn
Datasets: German Traffic Sign Dataset (GTSRB)
Libraries: NumPy, Pandas, Matplotlib, Seaborn
